Automatic Question paper generation Software

In today’s age, education is the most important way of achieving success. When we discuss education, it is imperative to mention tests and examination. Examinations prepare students in their quest for knowledge. So, having a proper examination paper and format is quite necessary.

Traditional method of generating question paper has been manual in which teacher’s chalk out the question paper on basis of their knowledge, experience and style. But this method can be ineffective at times as it can be time consuming and has repetition and security concerns.

Therefore, to address these concerns, our software solution provides an Automated process of Question Paper Generation which is fast, streamlined, randomized and secure.

The system takes over the entire difficult task and paperwork swiftly and efficiently.

The automated question paper generator solution performs all work related to question paper creation starting with

  1. Creating a question bank supporting multiple question types
  2. Defining the chapter weightage along with the difficulty level for the question paper
  3. Configure the questions distribution basis the question type and define the quantity and marks for every question type
  4. Avoid repetition by excluding questions from previously generated question papers
  5. Generate the question paper in split of a second.

Question Bank Creation

The generation of question paper starts with creation of a question bank. The created question bank is stored in a database and can be reused. Question bank is based on a course, subject, chapter and topic level there by giving complete granularity.

Support for adding different question types:

MCQ’s

Fill In the Blanks

True/False

Very Short Answer

Short Answer

Medium Answer

Long Answer

Essay

Comprehension

We provide an option to not only add questions manually but also support Bulk upload of your existing question banks as well.

We also provide an option to download the question bank for ease of use.

For every question added to the QB we give the provision to set the,

Difficulty Level: Easy / Medium / Hard

Category: Analytical / Theoretical / Logical/ Numerical

State: Active/Inactive

Author who added the question

Answer along with explanation if any

Automated Question paper generation

Question paper is generated based on course, subject, chapter and topic level there by giving full granularity. Analytics and dashboard are also provided for visibility and control.

As part of the generation process, the educator can configure the following

  1. Question paper name, Time and Total Marks
  2. Difficulty Distribution percentage
  3. Chapter selection along with weightage
  4. Questions distribution basis of question type
  5. Excluding questions from previously generated question papers.

This tool provides support for features like

  1. Question search
  2. Question entry formatting based on different type of questions
  3. View Question Bank summary
  4. View all questions generated using a single button and replace any question at ease and regenerate entire question paper again in seconds
  5. Export generated question paper to word file where you can specify your institute’s header and footer and watermark to brand it.

This tool will be useful for both small and large education institutes as the authorities will have the flexibility to generate class test, unit test, terminal test and final test online/offline.

Since this entire process of question paper generation is randomized and automated there is minimal human intervention thereby increasing productivity.
